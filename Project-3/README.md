A To-Do List application built using Python that helps users organize and manage their daily tasks through a simple command-line interface. The application allows users to add, view, update, mark as complete, and delete tasks efficiently. This project demonstrates core Python programming concepts such as lists, loops, functions, conditional statements, user input handling, and file handling (if tasks are stored permanently). It is a beginner-friendly project designed to improve programming skills while creating a practical productivity tool.
