A Python-based Command Line Number Guessing Game where the player tries to guess a randomly generated number within a specified range. The game provides hints after each incorrect guess, indicating whether the guessed number is too high or too low, making the gameplay interactive and engaging. This project demonstrates the use of Python fundamentals such as variables, loops, conditional statements, user input handling, the random module, and basic game logic. It is a beginner-friendly project designed to strengthen problem-solving skills and understand core programming concepts through a fun and interactive application.
